Kako koristiti API ključ

                        
Za slanje API zahtjeva uključite svoj API ključ u parametre upita, ovisno o zahtjevima
                            API-ja.
                        

                        
Evo primjera korištenja API ključa u Curl zahtjevu:

                        
                            curl -X POST \
                            -H "Content-Type: application/json" \
                            -d '{
                                {{podaci}}
                            }' \
                            "http://localhost:3000/api/foiPodaci/dodajPodatak?apiKljuc=YOUR_API_KEY_HERE"
                    

                        
Zamijenite YOUR_API_KEY_HERE svojim stvarnim API ključem u zahtjevu.
